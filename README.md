# Poet's Assistant — приложение для поэтов

**Poet's Assistant** — мобильное приложение на Kotlin, созданное для помощи тем, кто хочет научиться писать стихи.  
Разработано в рамках дипломного проекта в **РТУ МИРЭА**.

---

## Возможности

- Ведение заметок с форматированием 
- Библиотека статей о поэзии
- Генератор рифм для русских слов

---
